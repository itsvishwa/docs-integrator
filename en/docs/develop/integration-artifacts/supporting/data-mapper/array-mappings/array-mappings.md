# Array Mappings

When the source or target field is an array, the data mapper offers dedicated options for the array shape you need. The available choices depend on whether the mapping is array-to-array or array-to-single-value.

| Topic | Description |
|---|---|
| [Array to array](./array-to-array.md) | Map each element, assign as is, nested iterate, or join two arrays on a condition. |
| [Array to single value](./array-to-single-value.md) | Extract a single element, or aggregate the array into one value. |

- [Array to single value](./array-to-single-value.md) — Reduce an array to a single value.
