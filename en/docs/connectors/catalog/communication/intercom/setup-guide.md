# Setup Guide

This guide walks you through creating an Intercom account and obtaining the API access token required to use the Intercom connector.

## Step 1: Create an Intercom account

1. Navigate to the [Intercom website](https://www.intercom.com/) and sign up or log in.
2. Ensure you have a paid plan (Starter or higher) — API access is not available on the free trial.

## Step 2: Generate an API access token

1. Log in to your Intercom account.
2. Navigate to **Settings** (gear icon in the top-right corner) and select **For Developers** from the left sidebar.
3. In the Developer Hub, select **Your Apps**, then select **New app** or choose an existing app.
4. Go to the **Configure** tab and scroll down to the **Access Token** section to generate your token.

:::tip
Copy and store the token immediately — it will not be visible again in your account settings.
:::
