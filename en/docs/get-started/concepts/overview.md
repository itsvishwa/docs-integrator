# Concepts

The WSO2 Integration Platform has two sides to it: designing and building integrations in the WSO2 Integrator IDE, and deploying, running, and managing them in WSO2 Cloud. Understanding the vocabulary for each side helps you navigate the platform and make informed architecture decisions. The concepts in this section are organized the same way.

## Core concepts

[Core concepts](core.md) covers everything you work with when designing and building integrations in the IDE: organizations, projects, integrations, endpoints, and connections, along with the building blocks you compose integrations from.

## WSO2 Cloud concepts

[WSO2 Cloud concepts](integration-cloud-concepts.md) covers the managed runtime: data planes, environments, deployment tracks, and CI/CD. These concepts apply when you deploy integrations to WSO2 Cloud.
